# Contributing to AI Code Review Automator

Thank you for considering contributing to AI Code Review Automator! This document provides guidelines and instructions for contributing.

## Code of Conduct

Please be respectful and considerate of others when contributing to this project.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tab
- Include a clear description and steps to reproduce
- Include environment details (Node.js version, OS, etc.)

### Suggesting Enhancements
- Use the GitHub Issues tab
- Describe the enhancement and why it would be useful
- Include examples if possible

### Code Contributions
1. Fork the repository
2. Create a new branch for your feature/fix
3. Write tests for your changes
4. Ensure linting passes
5. Submit a pull request

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Build the project: `npm run build`
4. Run tests: `npm test`

## Pull Request Process

1. Update the README.md if needed
2. Add tests for new functionality
3. Ensure all tests pass
4. Update documentation
5. Request review from maintainers
